# One Piece Memory Match

A themed JavaScript memory match game.

---
## Technologies Used

* JavaScript
* jQuery
* HTML5
* CSS3
---
## Live Demo

[Live](http://memorymatch.vn-codes.com/)

---
## Features

* User can flip two cards to match them.
* User can see their game statistics.
